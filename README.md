## 📌 Overview

This project recreates the YouTube homepage UI to demonstrate frontend
development skills — including responsive grid layouts, component structure,
hover interactions, and clean semantic HTML. Built without any CSS frameworks
to show deep understanding of core web technologies.

## ✨ Features

- Responsive video card grid (adapts to mobile, tablet, desktop)
- Sticky header with search bar and icon buttons
- Sidebar navigation with category filters
- Video thumbnails with channel avatar, title, views and time
- Hover effects on cards and sidebar items
- Dark/light mode toggle (if implemented)

## 🚀 How to Run

**Option 1 — Open directly**
```bash
git clone https://github.com/BhumikaBC07/youtube-clone-ui
cd youtube-clone-ui
open youtube.html
```

**Option 2 — Live Server (VS Code)**
- Install the "Live Server" extension in VS Code
- Right-click `youtube.html` → **Open with Live Server**
- Runs at `http://127.0.0.1:5500`

**Option 3 — Live Demo**
Visit the deployed version → [Click Here](http://127.0.0.1:8001/youtube.html)
